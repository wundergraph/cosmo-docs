---
title: "Subgraph"
icon: circle-info
description: "Subgraphs are the building blocks of a federated graph. Each subgraph in a Federated GrapQL Architecture is responsible for defining its own part of the schema and handling the corresponding queries."
sidebarTitle: Overview
---

Before you can interact with a subgraph, you need to [`create`](/cli/subgraph/create) it. Each subgraph is mapped to a GraphQL service that can respond to the relevant GraphQL schema.
