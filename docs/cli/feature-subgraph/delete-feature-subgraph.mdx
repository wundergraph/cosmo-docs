## Description

Delete a feature subgraph within the specified (otherwise "default") namespace. A feature subgraph is an "override" of an existing subgraph.

## Usage

The command to delete the feature subgraph is the same as the subgraph. Please refer to [subgraph delete](/cli/subgraph/delete).
